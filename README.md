# 🎸 Chord & Scale Trainer

マイクで答え合わせするギターコード／スケール練習アプリ（ジャズ仕様・完全オフラインPWA）。

## 特徴
- コード／スケールを問題形式で出題し、**マイクで拾った演奏を自動判定**
- コード名＋ルート弦（6弦/5弦）だけ出題 → 弾く → 正誤に関わらず答えのダイアグラム表示 → 次へ
- **キー（ダイアトニック）出題**：キーを決めると ii-V-I の世界だけが出題される
- **5度圏ファンクションホイール**（2重リング・平行調配置）：出題中のコードの機能を可視化
- **索引モード**：全コード・スケールを使い所コメント＋ダイアグラム付きで一覧。スケールは
  役割で色分け（コードトーン／特徴音／アボイド）
- **苦手優先モード**・コード別/スケール別の**正答率記録**（localStorage、JSON入出力可）
- 単一ページ＋Service Worker で**インストール後は完全オフライン動作**

## 使い方
1. HTTPS で配信し（GitHub Pages / Netlify など）、スマホの Safari / Chrome で開く
2. マイクの使用を許可
3. ホーム画面に追加すると、以後オフラインでアプリのように起動できる

※マイク（getUserMedia）は HTTPS でのみ動作します。`file://` で直接開くと入力は使えません。

## ローカルで動かす
```
python -m http.server 8137
# http://localhost:8137/index.html を開く（localhost は secure context なのでマイクも動く）
```

## ファイル構成
- `index.html` … アプリ本体（UI・音声解析・SVG描画すべて）
- `manifest.json` / `sw.js` … PWA（インストール・オフライン）
- `icon-192.png` / `icon-512.png` / `apple-touch-icon.png` … アイコン
- `仕様書.md` … 詳細仕様と決定事項ログ
